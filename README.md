## About

In this workshop that showcases how to manage concurrent task execution.

<br />

## Getting Started

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/developer-academy-unina/Workshop-Concurrency
   ```

2. Open the ```.xcodeproj``` file

### How to

- What is concurrency and what is the difference between asynchronous code;
- Understand the common goal and errors with concurrency;
- Understand the life of a task;
- Showcase a couple of different exemples on GDC, Actor and (intro to) Combine.

## Issues and Discussions

You've found a bug in the source code, a mistake in the documentation or maybe you'd like a new feature? Take a look at [GitHub Discussions](https://github.com/developer-academy-unina/Workshop-Concurrency/discussions) to see if it's already being discussed. You can help us by [submitting an issue on GitHub](https://github.com/developer-academy-unina/Workshop-Concurrency/issues). Before you create an issue, make sure to search the issue archive -- your issue may have already been addressed!

Please try to create bug reports that are:

- _Reproducible._ Include steps to reproduce the problem.
- _Specific._ Include as much detail as possible: which version, what environment, etc.
- _Unique._ Do not duplicate existing opened issues.
- _Scoped to a Single Bug._ One bug per report.
